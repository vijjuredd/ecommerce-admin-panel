# 🛒 E-Commerce Admin Panel

A full-stack **E-Commerce Admin Panel** built using **Java Full Stack** technologies. This application enables administrators to efficiently manage products, categories, customers, and orders through a user-friendly interface.

## 🚀 Features

* Admin Login & Authentication
* Dashboard Overview
* Product Management (Add, Update, Delete)
* Category Management
* Order Management
* Customer Management
* Responsive User Interface
* CRUD Operations
* REST API Integration

## 🛠️ Tech Stack

**Backend**

* Java
* Spring Boot
* Spring Data JPA
* Hibernate
* MySQL

**Frontend**

* React.js
* HTML5

* JavaScript
* Bootstrap

## ⚙️ Getting Started

### Prerequisites

* Java 17+
* Maven
* MySQL
* Node.js & npm

### Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/ecommerce-admin-panel.git
```

2. Configure the MySQL database.

3. Start the Spring Boot backend.

4. Start the React frontend.

```bash
npm install
npm start
```
